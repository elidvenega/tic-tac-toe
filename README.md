# Tic Tac Toe

A tic tac toe game build using React 
## Tech Stack

**Client:** React, CSS

## Demo

https://tic-tac-toegame1.netlify.app/

## Installation

```bash
  git clone https://github.com/elidvenega/tic-tac-toe.git
  # Remove the current origin repository
  git remote remove origin
```

```bash
  cd tic-tac-toe
  npm install
  npm run dev
```
